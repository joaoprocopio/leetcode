# leetcode

Todos os Test Cases usados foram pegos diretamente do site do LeetCode.

Para rodar os testes do projeto:

```sh
poetry install
poetry shell
pytest
```

Estrutura do projeto:

```sh
src
├── easy
│   ├── array_concatenation
│   ├── convert_temperature
│   ├── palindrome_number
│   ├── plus_one
│   ├── reverse_string
│   └── single_number
├── medium
│   ├── multiply_strings
│   └── reverse_integer
└── hard
    ├── largest_rectangle_in_histogram
    └── shortest_palindrome
```
